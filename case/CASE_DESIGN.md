# PwnZero 外壳设计说明

## 外壳尺寸
- **外形**: 85.6mm x 53.98mm (标准信用卡尺寸)
- **厚度**: 25mm
- **材质**: 3D打印 PLA/PETG

## 三层结构设计

```
┌─────────────────────────────┐
│  透明曲面面板 (2mm)          │  ← 顶层：水晶滴胶/亚克力热弯
├─────────────────────────────┤
│  PCB 扩展板                  │  ← 中层：定制PCB
│  - OLED                      │
│  - 五向按键                  │
│  - LED x3                    │
├─────────────────────────────┤
│  Pi Zero 2W + PiSugar 3      │  ← 底层：主控+电池
└─────────────────────────────┘
```
