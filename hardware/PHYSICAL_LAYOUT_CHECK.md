# PwnZero 元器件物理尺寸验证

## PCB尺寸
- 宽度: 85.6mm
- 高度: 53.98mm
- 可用区域: 约80mm x 49mm (扣除边缘5mm安全区)

---

## 底层元器件 (Bottom Layer)

### 1. Pi Zero 2W (J1)
- 位置: (42.8, 27)
- 尺寸: 65mm x 30mm (排针区域)
- 占用区域: X: 10-75mm, Y: 12-42mm
- **这是最大的元器件，其他元器件必须避开此区域**

### 2. CC1101 TX (U2)
- 位置: (11, 14)
- 尺寸: 约16mm x 27mm
- 占用区域: X: 3-19mm, Y: 0.5-27.5mm
- **与Pi Zero距离: 约8mm** ✅

### 3. CC1101 RX (U3)
- 位置: (11, 41)
- 尺寸: 约16mm x 27mm
- 占用区域: X: 3-19mm, Y: 27.5-54mm
- **与Pi Zero距离: 约8mm** ✅

### 4. PN532 (U4)
- 位置: (45, 27)
- 尺寸: 约43mm x 41mm
- 占用区域: X: 23.5-66.5mm, Y: 6.5-47.5mm
- **⚠️ 警告: 与Pi Zero重叠！**

### 5. SMD电阻 (R1-R4)
- 位置: (75, 5/13/21/35)
- 尺寸: 2mm x 1.25mm (0805封装)
- 占用区域: 很小，无冲突 ✅

### 6. SMD电容 (C1-C4)
- 位置: (8, 8/11/35/38)
- 尺寸: 2mm x 1.25mm (0805封装)
- 占用区域: 很小，无冲突 ✅

### 7. IR接收器 (U6)
- 位置: (78, 10)
- 尺寸: 约6mm x 6mm
- 占用区域: X: 75-81mm, Y: 7-13mm
- 无冲突 ✅

### 8. IR LED (D1)
- 位置: (78, 35)
- 尺寸: 5mm直径
- 占用区域: X: 75.5-80.5mm, Y: 32.5-37.5mm
- 无冲突 ✅

## ⚠️ 发现的冲突

### 冲突1: PN532与Pi Zero重叠
```
PN532: X: 23.5-66.5mm, Y: 6.5-47.5mm
Pi Zero: X: 10-75mm, Y: 12-42mm
重叠区域: X: 23.5-66.5mm, Y: 12-42mm
```

**解决方案: PN532应该移到Pi Zero旁边，而不是下面**

### 冲突2: OLED与Pi Zero Z轴重叠
```
OLED在顶层，Pi Zero在底层
如果PCB厚度1.6mm，间距不足
```

**解决方案: 需要确认OLED是否使用排针连接，留出足够高度**

## 建议的元器件重新布局

### 底层布局优化:
```
左侧区域 (X: 0-20mm):
- CC1101 TX (11, 14)
- CC1101 RX (11, 41)
- 电容 C1-C4 (8, 8/11/35/38)

中央区域 (X: 20-70mm):
- Pi Zero 2W (42.8, 27) - 保持不变

右侧区域 (X: 70-85mm):
- 电阻 R1-R4 (75, 5/13/21/35)
- IR接收器 (78, 10)
- IR LED (78, 35)

PN532需要移动到:
- 建议位置: (25, 5) - Pi Zero左上方
- 或者: (70, 27) - Pi Zero右侧
```

### 顶层布局优化:
```
左侧区域:
- OLED (31, 16) - 需要确认高度

中央区域:
- 摇杆 (60, 27)

右侧区域:
- LED D2-D4 (78, 5/13/21)
```
