# PwnZero 外壳设计说明

## 外壳尺寸
- **外形**: 85.6mm x 53.98mm (标准信用卡尺寸)
- **厚度**: 25mm
- **材质**: 3D打印 PLA/PETG

## 三层结构设计

```
┌─────────────────────────────┐
│  透明曲面面板 (2mm)          │  ← 顶层：水晶滴胶/亚克力热弯
├─────────────────────────────┤
│  PCB 扩展板                  │  ← 中层：定制PCB
│  - OLED                      │
│  - 五向按键                  │
│  - LED x3                    │
├─────────────────────────────┤
│  Pi Zero 2W + PiSugar 3      │  ← 底层：主控+电池
└─────────────────────────────┘
```

## 曲面面板制作方案

### 方案一：水晶滴胶 AB 胶

**优点：**
- 透明度高
- 自然形成曲面
- 成本低 (~¥20)

**制作步骤：**
1. 3D打印模具 (凹槽形状)
2. 混合 AB 胶 (1:1)
3. 倒入模具
4. 静置 24 小时固化
5. 脱模打磨

### 方案二：亚克力热弯

**优点：**
- 强度高
- 可重复加工
- 表面光滑

**制作步骤：**
1. 购买 2mm 透明亚克力板
2. 加热至 160°C
3. 压入模具成型
4. 冷却定型

### 方案三：光固化树脂

**优点：**
- 精度最高
- 透明度极佳
- 适合小批量

**制作步骤：**
1. 使用光固化 3D 打印机
2. 选择透明树脂
3. 直接打印曲面
4. UV 固化
