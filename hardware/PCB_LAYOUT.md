## PCB 布局计算

### 信用卡尺寸限制
- 可用面积: 85mm x 54mm
- Pi Zero 占用: 65mm x 30mm (中心位置)
- 剩余边缘: 左右各 10mm, 上下各 12mm

### 顶面布局 (用户界面层)
```
     0    10   20   30   40   50   60   70   80  85mm
  0  ┌────┬────────────────────────────────────┬────┐
     │    │                                    │    │
 10  │    │  [OLED 35.4x33.5]                 │LED │
     │    │                                    │RGB │
 30  │    ├────────────────────────────────────┤    │
     │    │                                    │    │
 40  │    │  [五向摇杆 30.2x26]                │    │
     │    │                                    │    │
 54  └────┴────────────────────────────────────┴────┘
```

### 底面布局 (功能模块层)
```
     0    10   20   30   40   50   60   70   80  85mm
  0  ┌────┬────────────────────────────────────┬────┐
     │    │                                    │    │
 10  │CC1 │  [PN532 40.4x42.7]                │IR  │
     │101 │                                    │RX  │
 30  │TX  ├────────────────────────────────────┤    │
     │    │                                    │    │
 40  │CC1 │  Pi Zero 连接区                    │IR  │
     │101 │  (40pin 排针)                      │TX  │
 54  └────┴────────────────────────────────────┴────┘
```
